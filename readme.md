# 🛡️ appJB-Backend (OAuth05 API)

Este es el **Backend** de la aplicación AppJB, desarrollado con **Django Rest Framework** y utilizando contenedores con **Docker**.

## 🚀 Servidor
El servidor estará escuchando en:
`http://localhost:8000`

## 📂 Tecnologías
* **Lenguaje:** Python
* **Framework:** Django & Django Rest Framework
* **Base de Datos:** MySQL 8.4
* **Infraestructura:** Docker & Docker Compose

## 🛠️ Cómo levantar el proyecto
Para correr este Backend en tu PC, solo necesitas ejecutar:

```bash
docker-compose up --build
